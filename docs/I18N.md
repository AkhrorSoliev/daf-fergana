# Tarjima tizimi (i18n)

## Umumiy ko'rinish

Loyihada **custom i18n yechim** ishlatiladi (tashqi kutubxonasiz).

- **Tillar**: O'zbek (`uz`), Nemis (`de`)
- **Saqlash**: `localStorage["locale"]`
- **Default**: `uz`
- **Yo'l (URL)**: tilga qarab URL o'zgarmaydi — faqat kontent o'zgaradi

## Arxitektura

```
src/i18n/
├── I18nProvider.tsx          # React Context provider ("use client")
├── dictionaries.ts           # Re-export + getFromDict helper
└── dictionaries/
    ├── uz.ts                 # O'zbek tarjimalar (~260 qator)
    └── de.ts                 # Nemis tarjimalar (~260 qator)
```

## I18nProvider

**Fayl**: `src/i18n/I18nProvider.tsx`

```typescript
// Context tipi
type I18nContextValue = {
  locale: Locale;                          // "uz" | "de"
  t: (path: string) => string;            // Tarjima funksiyasi
  setLocale: (locale: Locale) => void;     // Tilni o'zgartirish
};

// Hook
export function useI18n(): I18nContextValue;
```

### Boshlang'ich til aniqlash

1. `localStorage["locale"]` tekshiriladi
2. `navigator.language` / `navigator.languages` tekshiriladi:
   - `de*` → Nemis
   - `uz*` → O'zbek
   - `ru*` → O'zbek (default)
3. Default: `"uz"`

### Hydration xavfsizligi

- Server va birinchi client renderda `"uz"` ishlatiladi
- `useEffect` bilan `detectInitialLocale()` chaqiriladi (mount da)
- `suppressHydrationWarning` root `<html>` da

## Tarjimalardan foydalanish

### Client komponentlarda

```typescript
"use client";
import { useI18n } from "@/i18n/I18nProvider";

function MyComponent() {
  const { t, locale, setLocale } = useI18n();

  return (
    <div>
      <h1>{t("hero.subtitle")}</h1>
      <p>{t("consultation.form.name")}</p>
      {locale === "de" && <span>German-specific content</span>}
    </div>
  );
}
```

### Dot-notation yo'l

```typescript
t("consultation.form.name")
// → dictionaries[locale].consultation.form.name
// → "Ism Familiya *" (uz) yoki "Vor- und Nachname *" (de)
```

Agar kalit topilmasa, bo'sh string `""` qaytaradi.

## Tarjima kalitlari

### Asosiy bo'limlar

| Prefix | Bo'lim | Ishlatilgan joy |
|--------|--------|-----------------|
| `hero.*` | Hero section | HeroSection |
| `teachers.*` | O'qituvchilar | TeachersPage, TeacherDetail |
| `advantages.*` | Afzalliklar | AdvantagesSection |
| `media.*` | Media/galeriya | MediaSection |
| `contacts.*` | Kontaktlar | (umumiy) |
| `about.*` | Markaz haqida | AboutSection, MediaSection |
| `consultation.*` | Konsultatsiya forma | ConsultationSection |
| `navbar.*` | Navigatsiya | Navbar |
| `blog.*` | Blog | BlogPage, BlogPost |
| `library.*` | Kutubxona | LibraryPage |
| `footer.*` | Footer | Footer |
| `contactsSection.*` | Manzillar bo'limi | ContactsSection |
| `courses.*` | Kurslar | CoursesPage |
| `lang.*` | Til nomlari | ThemeToggle |
| `metadata.*` | SEO meta | Layout files |

### Ko'p ishlatiladigan kalitlar

```
navbar.home          → "Asosiy sahifa" / "Startseite"
navbar.teachers      → "O'qituvchilar" / "Lehrer"
navbar.courses       → "Kurslar" / "Kurse"
navbar.cta           → "Bepul konsultatsiya" / "Kostenlose Beratung"

consultation.form.name           → "Ism Familiya *"
consultation.form.phone          → "Telefon raqam *"
consultation.form.submit         → "Konsultatsiya so'rang"
consultation.form.sending        → "Yuborilmoqda..."
consultation.form.success        → "Muvaffaqiyatli yuborildi!"
consultation.form.error          → "Xatolik yuz berdi."
consultation.form.phoneFormatError → "Telefon raqam formati: +998 XXX XX XX XX"
```

## Yangi tarjima qo'shish

1. `src/i18n/dictionaries/uz.ts` ga O'zbek tarjimasini qo'shing
2. `src/i18n/dictionaries/de.ts` ga Nemis tarjimasini qo'shing
3. Komponentda `t("yangi.kalit")` bilan foydalaning

**Misol:**
```typescript
// uz.ts
export const uz = {
  // ...mavjud kalitlar
  newSection: {
    title: "Yangi bo'lim",
    description: "Tavsif",
  },
};

// de.ts
export const de = {
  // ...mavjud kalitlar
  newSection: {
    title: "Neuer Abschnitt",
    description: "Beschreibung",
  },
};

// Komponentda:
t("newSection.title")  // → "Yangi bo'lim" yoki "Neuer Abschnitt"
```
